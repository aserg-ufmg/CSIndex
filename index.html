CSIndexbr provides transparent and meaningful data about the Brazilian scientific production in Computer Science. We index full papers published by Brazilian researchers in selected conferences and journals, since 2018. The papers are retrieved from DBLP. To report missing papers or other questions, use this form.

We are listed by GOTO Rankings * Our code and data are publicly available on GitHub * Last update: December 18, 2022.
